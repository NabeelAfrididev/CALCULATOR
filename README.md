# CALCULATOR
<P>This is my second Project But it is very simple calculator </P>

<h2>Overview</h2>

This is a simple, responsive Calculator Web Application built using HTML, CSS, JavaScript, and Bootstrap.
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

The project focuses on clean UI, responsive layout, and smooth button functionality using JavaScript event listeners.

<h2>Features</h2>

Responsive design using Bootstrap Grid System
Modern color theme with soft background
Real-time input display
Functional clear (“C”) and equals (“=”) buttons
Safe calculation using eval() for expression evaluation
